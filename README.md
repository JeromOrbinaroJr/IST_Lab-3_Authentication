# IST Lab 3 Authentication

Веб-приложение на Flask для лабораторной работы по аутентификации, ролям и разграничению доступа.

## Что реализовано

- Авторизация пользователей через логин и пароль.
- Ролевая модель с иерархией пользователей:
  `student1 -> starosta1 -> teacher1 -> curator1 -> dean1 -> admin`.
- Базовые разделы приложения:
  - главная страница профиля;
  - управление студентами;
  - записи с ACL-доступом;
  - роли и права;
  - пользователи для администратора;
  - учебный план.

## Локальное представление

Добавлена вкладка **"Учебный план"**.

Она реализует локальное представление из предметной области преподавателей, кафедр, дисциплин, квалификаций и назначений. Это не табличная ER-схема, а полноценный продуктовый экран:

- сводка по преподавателям, кафедрам, дисциплинам и часам;
- карточки кафедр;
- список дисциплин по кафедрам;
- карточки преподавателей;
- назначение, ставка и учебная нагрузка;
- матрица готовности преподавателей.

## Доступ по ролям

Для пользователей `student1`, `starosta1`, `teacher1` и `curator1` вкладка **"Учебный план"** доступна только для просмотра.

Для пользователей `dean1` и `admin` в этой вкладке появляется **админ-панель**, где можно редактировать:

- кафедры;
- дисциплины;
- преподавателей;
- должности преподавателей;
- даты назначений;
- ставки.

Изменения сохраняются в файл:

```text
instance/faculty_plan.json
```

Если файл ещё не создан, приложение использует данные по умолчанию из `app/faculty.py`.

## Основные файлы

- `app/faculty.py` - логика учебного плана, данные по умолчанию, права доступа и сохранение изменений.
- `templates/faculty/index.html` - интерфейс учебного плана и админ-панели.
- `templates/base.html` - пункт меню "Учебный план".
- `static/styles.css` - стили для нового продуктового экрана.
- `app/__init__.py` - регистрация blueprint учебного плана.

## Запуск

Установить зависимости:

```bash
pip install -r requirements.txt
```

Инициализировать базу данных:

```bash
python scripts/init_db.py
python scripts/seed_db.py
```

Запустить приложение:

```bash
python app.py
```

После входа в систему откройте вкладку **"Учебный план"**.
